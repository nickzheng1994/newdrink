# 牛马补水站

一个 macOS 菜单栏喝水提醒 App，写给认真搬砖但经常忘记喝水的打工人。

## 下载

前往 GitHub Releases 下载最新版本：

https://github.com/nickzheng1994/newdrink/releases/latest

下载 `牛马补水站-macOS.zip` 后解压，双击 `牛马补水站.app` 即可运行。

## 功能

- 顶部菜单栏水滴图标，点击后展开水杯面板
- 默认工作时间：09:00 到 17:30
- 工作时间、午休时间、提醒间隔和每日目标会保存到本地
- 默认每日喝水目标：2000 ml，最高可设置到 5000 ml
- 首次启动展示欢迎页，并引导设置工作时间、午休时间、提醒间隔和每日目标
- 可选择提醒间隔：15 分钟、30 分钟或 60 分钟
- 默认午休时间：11:30 到 13:30，午休期间不会提醒
- 到点后直接弹出置顶提醒窗口，不使用系统通知
- 提醒时可选择本次喝水量：100 ml 到 550 ml，50 ml 为间隔
- 点击喝水量选项即记录完成，不需要再点完成按钮
- 面板显示当前已喝水量，可拖拽滑块手动调整
- 滑块范围 0 到当前每日目标，100 ml 为一个单位
- 每次完成喝水都会记录一条本地事件，手动调整会记录差值
- 顶部面板默认显示今天每小时喝水图表，可切换查看本周
- 下班时间主动弹窗统计今天每个小时喝了多少水
- 距离下班前 30 分钟隐藏提醒手机充电，带插头插入手机接口动效
- 喝水提醒、下班统计和手机充电提醒都使用无边框液态玻璃弹窗
- 顶部面板提供“清缓存并退出”，方便测试首次进入流程

## 构建

需要 macOS 和 Xcode Command Line Tools。

```bash
./build_app.sh
```

构建完成后运行：

```bash
open ".build/牛马补水站.app"
```

## 本地数据

设置、喝水记录和首次进入状态保存在 `NSUserDefaults`，Bundle ID 为：

```text
com.ehousechina.drinktime
```
